# packman

Packman (resource PACK MANager) is a library plugin that manages the servers custom resource pack and custom items.

It allows server owners to add custom items, glyphs, etc to the game.

It allows plugin developers to write plugins that can add their own textures, items and glyphs into the game.

### Features

 - Generates a Minecraft Resourcepack
 - Hosts the resourcepack on a HTTPServer.
